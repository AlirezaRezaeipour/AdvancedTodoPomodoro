# Advanced Todo & Pomodoro

A Tkinter-based productivity app combining a ToDo list and a Pomodoro timer.

## Features
- Add, edit, delete, and track tasks
- Individual task timers for each task
- Pomodoro timer (25 minutes)
- Time remaining today with selectable country/timezone
- History tracking of total time spent
